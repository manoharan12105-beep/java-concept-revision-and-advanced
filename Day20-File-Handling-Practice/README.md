# Day 20 — File Handling Practice (Java)

This folder contains Java programs that practice **File Handling concepts** such as reading, writing, modifying, and analyzing file data.

## Topics Covered

* Reading files using `BufferedReader`
* Writing files using `BufferedWriter`
* Copying file content
* Processing text data in files
* Basic file manipulation

## Programs Included

### 1. CopyFileContent.java

Copies the content from one file to another using `BufferedReader` and `BufferedWriter`.

### 2. CountWordsInFile.java

Reads a file and counts the total number of words.

### 3. LongestLineInFile.java

Finds and prints the longest line in a file.

### 4. ReplaceWordInFile.java

Replaces a specific word in a file and writes the updated content to a new file.

### 5. CountVowelsInFile.java

Counts the total number of vowels present in a file.

## Concepts Practiced

* File input and output in Java
* Using buffered streams for efficient reading and writing
* String processing from file data
* Exception handling with `IOException`
