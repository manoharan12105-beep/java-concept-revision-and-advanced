# Day 19 — File Handling Basics (Java)

This project demonstrates basic **File Handling in Java** using `BufferedWriter` and `BufferedReader`.

The program writes data to a file and then reads the same file line by line.

## Concepts Covered

- File writing using `FileWriter`
- Efficient writing using `BufferedWriter`
- File reading using `FileReader`
- Line-by-line reading using `BufferedReader`
- Exception handling in file operations

## Program Description

The program performs the following steps:

1. Creates or overwrites a file called **Output.txt**
2. Writes a message and a list of friend names to the file
3. Reads the file using `BufferedReader`
4. Prints the file content to the console

## Example Output

```
Accessing the file.
My friends are:
     Doni
     Manikandan
     Nanda Kishore
     Nikil
     Nithish Kumar
```

## Technologies Used

- Java
- `java.io` package
- BufferedReader
- BufferedWriter
- FileReader
- FileWriter

## Learning Outcome

This example demonstrates how Java programs can **store and retrieve data from files**, an important concept used in logging systems, configuration files, and backend applications.
