# Day 18 – Exception Handling (Java)

This folder contains Java programs demonstrating core Exception Handling concepts in Java.

These examples focus on understanding how runtime errors occur, how they propagate, and how to handle them properly.

---

## 📂 Files Included

✔ BasicTryCatch.java  
Demonstrates a simple try–catch block handling ArithmeticException.

✔ MultipleCatchExample.java  
Shows how to use multiple catch blocks and why specific exceptions should come first.

✔ FinallyExample.java  
Illustrates how the finally block always executes and is used for cleanup logic.

✔ ThrowExample.java  
Demonstrates manually throwing an exception using the throw keyword.

✔ ExceptionPropagationExample.java  
Shows how exceptions propagate through the call stack until they are handled.

---

## 🧠 Concepts Covered

- What is an Exception?
- try–catch block
- Multiple catch blocks
- finally block
- throw keyword
- Exception propagation
- Runtime exceptions
