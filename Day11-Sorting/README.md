# Day 11 — Sorting Algorithms

This folder contains Java implementations of the following sorting algorithms:

⭐ Bubble Sort  
⭐ Selection Sort  
⭐ Insertion Sort  
⭐ Quick Sort  
⭐ Merge Sort

Each file contains a runnable `main` method to accept input from the user and print the sorted array.

### Time Complexities

| Algorithm       | Best   | Avg   | Worst  | Space | Stable |
|----------------|--------|-------|--------|--------|--------|
| Bubble Sort     | O(n)   | O(n²) | O(n²) | O(1)  | Yes    |
| Selection Sort  | O(n²)  | O(n²) | O(n²) | O(1)  | No     |
| Insertion Sort  | O(n)   | O(n²) | O(n²) | O(1)  | Yes    |
| Quick Sort      | O(n log n) | O(n log n) | O(n²) | O(log n) | No |
| Merge Sort      | O(n log n) | O(n log n) | O(n log n) | O(n) | Yes |
